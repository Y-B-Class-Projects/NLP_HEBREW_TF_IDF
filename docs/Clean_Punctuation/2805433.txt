חברי הכנסת תמר זנדברג  ( מרצ )  ואיציק שמולי  ( העבודה )  הוצאו היום  ( רביעי )  ממליאת הכנסת בזמן נאום ראש הממשלה בנימין נתניהו לאחר שהתפרצו לדבריו .  מאוחר יותר הוצאו מהמליאה גם ח"כ מסעוד גנאים  ( רע"מ-תע"ל )  וחבר הכנסת אראל מרגלית  ( העבודה )  .  בזמן נאומו של יו"ר האופוזיציה יצחק הרצוג הוצאה מהאולם ח"כ מירי רגב  ( הליכוד )  .
