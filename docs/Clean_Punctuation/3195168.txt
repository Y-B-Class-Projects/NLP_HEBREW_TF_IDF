ח"כ פרץ דורש לזמן את רה"מ להייתחס להכרזה הירדנית
ח"כ עמיר פרץ  ( המחנה הציוני )  שלח היום  ( שני )  מכתב ליו"ר ועדת החוץ והביטחון אבי דיכטר בו הוא דורש לזמן את ראש הממשלה לועדת החוץ והביטחון כדי  " להתייחס לפעולות שננקטו טרם ההכרזה הירדנית על אי הארכת הנספח ".
