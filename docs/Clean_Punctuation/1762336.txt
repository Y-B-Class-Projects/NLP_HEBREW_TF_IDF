ח"כ כץ בנאום לבני בכנסת : " תנו לה טישו ,  שלא תבכה "
 חבר הכנסת חיים כץ  ( ליכוד )  העיר בצהריים  ( רביעי )  במהלך נאומה של יו"ר האופוזיציה ציפי לבני  ( קדימה )  בכנסת הערת ביניים : " אפשר לתת לה טישו ?  היא עומדת לבכות ".  בתגובה כינה אותו ח"כ רוני בראון  ( קדימה )   " סקסיסט עלוב ",  ואילו ח"כ נחמן שי  ( קדימה )  אמר כי  " ההתבטאות הזאת היא התבטאות מעליבה ".
